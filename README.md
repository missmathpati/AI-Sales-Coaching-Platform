## Shipped System: AI Sales Coaching Platform
**Blue AI Labs · 2025 · Proprietary**

End-to-end LLM inference pipeline that transforms sales call transcripts into structured coaching reports.

**Problem:** Sales managers lacked scalable, consistent feedback on rep performance.

**What I built:**
- Transcript ingestion and preprocessing pipeline (FastAPI + Supabase)
- Multi-call context weighting for LLM-based analysis (LangChain + Claude API)
- Structured XML report generation with schema validation
- Drift detection and reproducibility controls across prompt versions

  **Results:** 82% alignment to expert-written reports · 60% reduction in manual review time · 30% reduction in output errors

**Evaluation methodology:** Expert-alignment scoring, bias checks on demographic edge cases, schema drift monitoring across 1,000+ inferences

**Stack:** Python, FastAPI, Supabase, LangChain, Claude API, Docker
